# AeonGYM — Final Project

Overview

AeonGYM is a responsive, multi-page website designed to embrace gym culture and support users in fitness and nutrition. The platform provides workout programs, nutrition tracking and community interaction, combining functionality with a clean and modern design.

## Features
Pages: Home, Training, Nutrition, Community, About
Interactive Tools:
Calorie & macronutrient calculator
Exercise selection and weekly plan generator
Community Engagement:
Contact form with input validation
Review table for community feedback
Design & Layout:
Responsive across desktop and mobile devices
Uses Bootstrap 5 grid system, Flexbox, and CSS Grid
Smooth animations and visual feedback for interactive elements

## Technologies Used
HTML5
CSS (Flexbox & Grid)
Bootstrap
JavaScript
jQuery
